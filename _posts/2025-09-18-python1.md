### 문제 설명

약수의 개수가 세 개 이상인 수를 합성수라고 합니다. 자연수 `n`이 매개변수로 주어질 때 `n`이하의 합성수의 개수를 return하도록 solution 함수를 완성해주세요.



### 제한사항

- 1 ≤ `n` ≤ 100



### 입출력 예시

| n | result |
| --- | --- | 
| 10 | 5 |
| 15 | 8 |


### 내 풀이

```python
import math
def solution(n):
    answer = 0

    for i in range(1, n+1):
        cnt = 0
        for j in range(1, int(math.sqrt(i))+1):
            if (i%j==0 and j==(i//j)):
                cnt += 1
            elif (i%j==0):
                cnt += 2
        if (cnt >= 3):
            answer += 1
    return answer
```

- 실행 시간: 최소 0.01ms, 최대 0.18ms
- 메모리 사용량: 최소 9.16MB, 최대 9.36MB



### 코드 설명
math.sqrt(i)
