### 문제 링크

[문제 바로가기](https://school.programmers.co.kr/learn/courses/30/lessons/132202)



### 내 풀이
MYSQL
```sql
SELECT MCDP_CD AS 진료과코드, 
       COUNT(*) AS 5월예약건수 
FROM APPOINTMENT
WHERE APNT_YMD LIKE "2022-05%"
GROUP BY MCDP_CD
ORDER BY 5월예약건수 ASC, 진료과코드 ASC;
```

<br>

### 다른 풀이
```sql
SELECT MCDP_CD AS "진료과코드",
       COUNT(PT_NO) AS "5월예약건수" 
FROM APPOINTMENT
WHERE APNT_YMD >= TO_DATE('2022-05-01', 'YYYY-MM-DD')
  AND APNT_YMD <  TO_DATE('2022-06-01', 'YYYY-MM-DD') 
GROUP BY MCDP_CD
ORDER BY "5월예약건수" ASC, "진료과코드" ASC;
```

<br>

### 기록

- TIMESTAMP 타입에 문자열 패턴 매칭을 사용하게 되면 데이터베이스는 모든 TIMESTAMP를 문자열로 변환하는 과정을 거침 (Full Table Scan)
- 날짜/시간 데이터는 범위조건으로 검색해야 인덱스를 탈 수 있음
