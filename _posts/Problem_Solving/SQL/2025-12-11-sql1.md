### 내 풀이
MYSQL
```sql
SELECT
  CASE
    WHEN (
      day = 'Sat' OR day = 'Sun'
    ) THEN 'weekend'
    ELSE 'weekday'
  END AS week,
  SUM(total_bill) AS sales
FROM
  tips
GROUP BY
  week
ORDER BY
  sales DESC;
```



### 다른 풀이
MYSQL
```sql
SELECT
  CASE
    WHEN day IN ('Sat', 'Sun') THEN 'weekend'
    ELSE 'weekday'
  END AS week,
  SUM(total_bill) AS sales
FROM
  tips
GROUP BY
  week
ORDER BY
  sales DESC;
```
- 가독성을 위해 IN () 사용
- 가급적 작은따옴표 사용 



### 기록

- 표준 SQL(ANSI SQL)"을 엄격하게 따를 경우 SELECT 문에서 정의한 변수를 GROUP BY에 사용할 수 없음. MySQL은 허용해주지만, 그렇지 않은 경우에는 1) SELECT문에 작성한 CASE WHEN 구문을 그대로 GROUP BY에 작성 2) WITH 구문으로 미리 변수를 생성한 뒤 GROUP BY 작성
