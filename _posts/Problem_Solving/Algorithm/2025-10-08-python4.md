### 문제 설명

정수 리스트 num_list와 정수 n이 주어질 때, n 번째 원소부터 마지막 원소까지의 모든 원소를 담은 리스트를 return하도록 solution 함수를 완성해주세요.



### 제한사항

- 2 ≤ num_list의 길이 ≤ 30
- 1 ≤ num_list의 원소 ≤ 9
- 1 ≤ n ≤ num_list의 길이



### 입출력 예시


| num_list | is_nprefix | result | 
| --- | --- | --- |
| [2, 1, 6] | 3 | [6] | 
| [5, 2, 1, 7, 5] | 2 | [2, 1, 7, 5] | 


### 내 풀이

```python
def solution(num_list, n):
    return num_list[n-1:]
```
