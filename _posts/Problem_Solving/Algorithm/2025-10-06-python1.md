### 문제 설명

정수가 담긴 리스트 `num_list`가 주어질 때, 모든 원소들의 곱이 모든 원소들의 합의 제곱보다 작으면 1을 크면 0을 return하도록 solution 함수를 완성해주세요.



### 제한사항

- 2 ≤ `num_list`의 길이 ≤ 10
- 1 ≤ `num_list`의 원소 ≤ 9



### 입출력 예시


| num_list | result | 
| --- | --- | 
| [3, 4, 5, 2, 1] | 1 | 
| [5, 7, 8, 3] | 0 | 



### 내 풀이

```python
def solution(num_list):
    product = 1
    squared = 0
    
    for num in num_list:
        product *= num
        
    for num in num_list:
        squared += num
    squared = squared**2    
    
    
    if product < squared:
        return 1
    else:
        return 0
```
