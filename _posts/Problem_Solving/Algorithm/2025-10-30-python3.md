### 문제 설명

정수로 이루어진 문자열 n_str이 주어질 때, n_str의 가장 왼쪽에 처음으로 등장하는 0들을 뗀 문자열을 return하도록 solution 함수를 완성해주세요.



### 제한사항

- 2 ≤ n_str ≤ 10
- n_str이 "0"으로만 이루어진 경우는 없습니다.



### 입출력 예시


| n_str | result | 
| --- | --- |
| "0010" | "10"" | 
| "854020" | "854020" |



### 내 풀이

```python
def solution(n_str):
    idx = 0
    for i in range(len(n_str)):
        if n_str[i] != '0':
            idx = i
            break
        
    return n_str[i:]
```
