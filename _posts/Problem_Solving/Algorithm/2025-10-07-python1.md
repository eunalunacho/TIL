### 문제 설명

정수 리스트 `num_list`가 주어질 때, 첫 번째로 나오는 음수의 인덱스를 return하도록 solution 함수를 완성해주세요. 음수가 없다면 -1을 return합니다.





### 제한사항

- 5 ≤ `num_list`의 길이 ≤ 100
- -10 ≤ `num_list`의 원소 ≤ 100



### 입출력 예시


| num_list | result | 
| --- | --- |
| [12, 4, 15, 46, 38, -2, 15] | 5 |
| [13, 22, 53, 24, 15, 6] | -1 |



### 내 풀이

```python
def solution(num_list):
    for num in num_list:
        if num < 0:
            return num_list.index(num)
        
    return -1
```
